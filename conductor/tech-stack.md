# Technology Stack - DevQuest Bootcamp

## Frontend
- **HTML5:** Semantic markup for structure.
- **Tailwind CSS:** Utility-first CSS framework for rapid UI development and neon aesthetics.
- **TypeScript:** Typed JavaScript for enhanced development experience and maintainability.

## Build & Tooling
- **Vite:** Next-generation frontend tool for fast development and optimized production builds.
- **Node.js:** Runtime environment for development tools.

## Browser APIs
- **IntersectionObserver API:** Native browser API for scroll-triggered staggered reveal animations.
